# c2-calk, v1.0

# Калькулятор: журнал, кодировка, unit- и E2E-тесты
Браузерный калькулятор с журналом операций, защитой UTF-8 и автотестами (Vitest + Playwright).

# Среда разработки
- Cursor (Agent / Composer 2.5 Fast)
- Node.js v22.22.0, npm 11.12.1
- Git + GitHub (`evleonov-NS/c2-calk`)
- Playwright (Chromium, Firefox, WebKit)
- Vitest

# Что сделано
- Браузерный калькулятор в одном файле `calculator.html` (тёмная/светлая тема)
- Журнал операций (до 50 записей в `localStorage`, клик по записи подставляет результат)
- Защита UTF-8: `.editorconfig`, `.gitattributes` в проекте; глобально `~/.gitattributes`, `files.encoding: utf8` в Cursor
- Класс `Calculator` вынесен в `src/Calculator.js`
- Модульные тесты Vitest: 41 тест (`tests/Calculator.test.js`)
- E2E-тесты Playwright: 20 сценариев × 3 браузера = 60 прогонов (`e2e/calculator.spec.js`)
- HTML-отчёт Playwright (`npm run test:e2e:report`)

# Что изменилось
- **Функции:** ввод, `+ − × ÷`, `%`, `C`, `⌫`, запятая, цепочка операций, деление на 0 → «Ошибка», журнал, тема
- **Файлы:** `calculator.html`, `src/Calculator.js`, `tests/`, `e2e/`, `package.json`, `playwright.config.js`, `vitest.config.mjs`, `.editorconfig`, `.gitattributes`
- **Улучшено:** группировка E2E через `describe`, покрытие backspace / % / тема / очистка журнала, три браузера
- **Исправлено:** «крокозябры» (PowerShell `Set-Content`); случайно сломанный CSS (коммит «сломал»); конфликт Vitest ↔ Playwright (`vitest.config.mjs`)

# Изменения на сервере
- Не применимо (статический frontend, VPS не используется)
- GitHub: ветка `main`, force-push при откате истории (коммиты «сломал» и «Исправить CSS» удалены)

# Изменения в приложении
- **UI/UX:** двухколоночная вёрстка (калькулятор + журнал), переключатель темы, русская локаль чисел (запятая)
- **Логика:** state-машина калькулятора; запись в журнал после `=`
- **Компоненты:** монолит в `calculator.html` (UI + JS); логика дублирована/вынесена в `Calculator.js` для тестов
- **API:** нет
- **localStorage:** `calc-theme`, `calc-journal`

# Причина изменений
- Учебный проект: калькулятор с журналом и автотестами
- Защита кириллицы от порчи при работе в Windows/PowerShell
- Проверка поведения в браузере (E2E) и логики (unit) отдельно

- **Не использовать:** `git show ... | Set-Content` для файлов с кириллицей — ломает кодировку
